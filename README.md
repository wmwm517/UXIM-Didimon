# Didimon
디디몬같아🦖

> 자립준비청년을 위한 자립 로드맵  
> 지원사업 탐색, 로드맵, 투두리스트를 연결하는 자립 과정 관리 서비스

## Overview
Over the rainbow🌈

Didimon은 자립준비청년이 흩어진 자립 정보를 쉽게 찾고, 자신의 상황에 맞는 과제를 단계적으로 관리할 수 있도록 돕는 자립 로드맵 서비스입니다.

자립준비청년은 보호 종료 이후 주거, 금융, 학업, 취업, 건강, 법률, 생활 등 다양한 문제를 동시에 마주합니다. 그러나 기존 정보는 여러 기관과 채널에 흩어져 있고, 사용자가 자신의 상황에 맞는 지원을 찾고 관리하기 어렵습니다.

Didimon은 개인 맞춤형 지원사업 탐색, 분야별 자립 로드맵, 투두리스트 기능을 통해 자립 과정을 시각화하고 지속적으로 관리할 수 있도록 설계되었습니다.

## Problem

자립준비청년은 만 18세 이후 보호가 종료되면 사회로 나와 스스로 생계를 책임져야 합니다. 이 과정에서 다음과 같은 문제가 발생합니다.

- 심리적 압박과 정신건강 위기
- 사회적 지지망 및 멘토 부족
- 주거 빈곤과 환경 격차
- 경제적 지원과 행정 절차의 복잡성
- 지원사업 정보의 분산
- 개인 상황에 맞는 정보 탐색의 어려움
- 지속적인 관리와 피드백 체계 부족

기존 솔루션은 정보를 제공하는 데는 의미가 있지만, 자립 과정을 장기적으로 관리하고 사용자의 행동을 유도하는 데 한계가 있습니다.

## Existing Solutions & Limitations

### 오픈채팅방

오픈채팅방은 빠른 정보 공유가 가능하지만, 운영진에 의존하고 공지 구조가 불안정하며 원하는 정보를 저장하거나 다시 찾기 어렵습니다.

### 자립해냄

자립해냄은 주변 청소년쉼터 검색, 자립준비수준 자가진단, 자립지원 정보 등을 제공했지만 운영 종료 및 업데이트 부재의 한계가 있습니다.

### 자립정보ON

자립정보ON은 자립지원 정보 접근성과 검색을 제공하지만, 개인 맞춤형 알림이나 스크랩 기반 관리 기능이 부족하고 상담 인력과 참여율에 한계가 있습니다.

## Solution

Didimon은 자립 정보를 단순히 보여주는 것을 넘어, 사용자가 자신의 자립 과정을 직접 계획하고 관리할 수 있도록 돕습니다.

서비스는 다음 세 가지 UX 방향성을 중심으로 설계되었습니다.

### 1. 사용자 맞춤

온보딩에서 관심 분야와 현재 상황을 수집하고, 사용자의 조건에 맞는 지원사업과 행동을 추천합니다.

### 2. 자기주도성

사용자가 직접 할 일을 입력하거나 추천 행동을 선택해 자신의 계획을 구성할 수 있도록 합니다.

### 3. 가시성

자립 준비 과정을 분야별 로드맵과 진행률로 시각화하여 사용자가 현재 위치와 다음 행동을 쉽게 파악할 수 있도록 합니다.

## Key Features

### 1. 지원사업 공고 탐색

금융, 주거, 취업, 학업, 생활문화 등 다양한 분야의 자립 지원사업 공고를 한 곳에서 확인할 수 있습니다.

- 키워드 검색
- 분야별 필터링
- 개인 관심 분야 기반 추천
- 지원사업 상세 정보 확인
- 관심 공고 저장

### 2. 자립 로드맵

사용자의 자립 진행 상황을 분야별로 시각화합니다.

- 전체 자립 진행률 확인
- 금융, 주거, 취업, 학업 등 분야별 진행 상태 확인
- 핵심 목표와 세부 행동 확인
- 추천 행동을 투두리스트에 추가
- 진행 완료 여부 반영

### 3. 투두리스트

자립 준비 과정에서 해야 할 일을 일정과 함께 관리합니다.

- 오늘의 할 일 확인
- 직접 할 일 등록
- 추천 행동 추가
- 날짜, 카테고리, 우선순위 설정
- 알림 설정
- 완료 체크
- 로드맵 진행률과 연동

## Target Users

- 보호 종료를 앞둔 예비 자립준비청년
- 보호 종료 후 자립 초기 단계에 있는 청년
- 자립 지원 정보를 찾기 어려운 사용자
- 금융, 주거, 취업, 학업 등 여러 문제를 동시에 관리해야 하는 사용자
- 자립 과정에서 체계적인 계획과 피드백이 필요한 사용자

## User Research

Didimon은 관련 대상자와 현장 이해관계자 인터뷰를 통해 문제를 구체화했습니다.

인터뷰 대상에는 시설 보호 청소년 멘토링 관계자, 자립준비청년 출신 사회복지사, 자립지원 공익법인 관계자가 포함되었습니다.

리서치를 통해 다음 인사이트를 도출했습니다.

- 자립준비청년은 경제적 지원뿐 아니라 심리·실천적 문제를 함께 겪음
- 지원사업 정보는 많지만 개인 상황에 맞게 찾고 관리하기 어려움
- 지속적인 피드백과 동기부여 체계가 필요함
- 커뮤니티 기능은 안전 체계와 공적 신뢰도가 중요함
- 자립 과정은 단기 정보 제공보다 장기적 관리가 필요함

## UX Direction

Didimon은 자립준비청년이 안심하고 머물 수 있는 따뜻하고 친근한 공간을 지향합니다.

디자인 방향은 다음과 같습니다.

- 따뜻하고 밝은 톤
- 친근한 언어
- 가족처럼 가깝고 믿을 수 있는 소통 창구
- 성장과 기록을 시각적으로 확인하는 경험
- 복잡한 기능보다 핵심 기능 중심의 간결한 UI
